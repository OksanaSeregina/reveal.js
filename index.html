WHAT IS LODASH?

Lodash is a JavaScript library which provides utility functions for dealing with javascript objects and arrays, enhancing productivity and code readability.

WHY LODASH?

Lodash helps programmers write more concise and easier to maintain JavaScript code. Lodash contains tools to simplify programming with strings, numbers, arrays, functions and objects.

Lodashâ€™s modular methods are great for:

  • Iterating arrays, objects, & strings
  • Manipulating & testing values
  • Creating composite functions

Adding Lodash to a project

In a browser:

Using npm:

			
				$ npm i -g npm
				$ npm i --save lodash

In Node.js:

			
				// Load the full build.
				var _ = require('lodash');
				// Load the core build.
				var _ = require('lodash/core');
				// Load the FP build for immutable auto-curried iteratee-first data-last methods.
				var fp = require('lodash/fp');
				// Load method categories.
				var array = require('lodash/array');
				var object = require('lodash/fp/object');
				// Cherry-pick methods for smaller browserify/rollup/webpack bundles.
				var at = require('lodash/at');
				var curryN = require('lodash/fp/curryN');

Collection methods

map, reduce, filter, some, every, forEach (aka each), size, ...

_.map

			
				const data = [
				{ name: 'Patrick',age: '25',},
				{ name: 'John',age: '24',},
				{ name: 'Teresa',age: '26',}
				];

				_.map(data, 'name');// [ 'Patrick', 'John', 'Teresa' ]

Works with nested properties as well

			
				const data = [
				{ name: 'Patrick',age: '25',profile: {experience: 2} },
				{ name: 'John',age: '24',profile: {experience: 2}},
				{ name: 'Teresa',age: '26',profile: {experience: 4} }
				];

				_.map(data, 'profile.experience');// [ 2, 2, 4 ]

Object methods

assign, at, keys, forIn, keys, values, ...

Object manipulation

			
				var object = { 'a': [{ 'b': { 'c': 3 } }] };

				_.get(object, 'a[0].b.c');// => 3

				_.set(object, 'a[0].b.c', 4);
				console.log(object.a[0].b.c);// => 4

				_.has(object, 'a');// => true

_.toPairs(object)

			
				_.toPairs([{ 'a': 1, 'b': 2 }]);// => [['a', 1], ['b', 2]]

_.fromPairs(pairs)

			
				_.fromPairs([['a', 1], ['b', 2]]);// => { 'a': 1, 'b': 2 }

_.cloneDeep

			
				const users = [
        {
          name: 'Patrick', age: '25', profile: { experience: 3 }
        },
        {
          name: 'John', age: '24', profile: { experience: 2 }
        },
        {
          name: 'Teresa', age: '26', profile: { experience: 4 }
        }
      ];

      const usersClone = _.cloneDeep(users);
      usersClone[0].age = '27';

      usersClone[0].age;
      // 27
      users[0].age;
      // 25

Function chaining

_.chain

			
				const users = [
        {
          name: 'Patrick', age: 25, profile: { experience: 3 }
        },
        {
          name: 'John', age: 24, profile: { experience: 2 }
        },
        {
          name: 'Teresa', age: 26, profile: { experience: 4 }
        }
      ];

      _.chain(users)
      .map('age')
      .mean()
      .value();
      // 25
			
		  
            
              const users = [
              {
                name: 'Patrick', age: 25, profile: { experience: 3 }
              },
              {
                name: 'John', age: 24, profile: { experience: 2 }
              },
              {
                name: 'Teresa', age: 26, profile: { experience: 4 }
              }
            ];

            const multiplyByTwo = numbers => _.map(numbers, num => num * 2);

            _.mixin({ multiplyByTwo });

            _.chain(users)
            .map('age')
            .multiplyByTwo()
            .mean()
            .value();
            // 50
